# Maid-Empire-Remastered

Maid-Empire-Remastered is a remastered Unciv mod based on the concept of Maid Empire.

This project updates the original Maid Empire mod concept for newer Unciv versions, fixes deprecated unique definitions, reorganizes JSON files, and improves compatibility.
